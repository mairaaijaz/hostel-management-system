==================== WORKFLOW 1: User Authentication (Student & Warden) ====================

Description:
This workflow allows students and wardens to securely log into the hostel management system using their assigned IDs.

Steps:
1. User opens the hostel management system website.
2. User selects login type: Student or Warden.
3. User enters Student ID or Warden ID along with their password.
4. System validates the credentials.
5. Based on role, the user is redirected to the respective dashboard (Student or Warden).

Key Features:
- Role-based login
- Secure authentication
- Separate dashboards for Student and Warden

===========================================================================================

==================== WORKFLOW 2: Room Allocation & Hostel Services ====================

Description:
This workflow allows students to request and view room allocation, and access hostel services.

Steps:
1. Student logs in and opens the “Room Allocation” section.
2. Student submits a request for a room or checks their existing room.
3. System checks availability and records the request.
4. System notifies the student of room allocation or updates.
5. Students can also view hostel facilities, meal plans, and other services.

Key Features:
- Room request and allocation status
- View hostel services
- Notifications for allocation updates

===========================================================================================

==================== WORKFLOW 3: Warden Management Actions ====================

Description:
This workflow allows the warden to manage student room allocations, requests, and hostel records.

Steps:
1. Warden logs into the system dashboard.
2. Warden views all pending room requests submitted by students.
3. Warden approves or rejects requests.
4. Warden updates hostel information and services if needed.
5. System updates student records and notifies students about status changes.

Key Features:
- Manage student room requests
- Update hostel info and services
- Notifications sent to students
- Dashboard with all student and hostel records

===========================================================================================
